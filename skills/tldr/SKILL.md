---
name: tldr
description: How to write when TLDR mode is on. Load when the user instructs you to TLDR your responses.
---
When the user asks for TLDR, turn TLDR mode on from that point on. When TLDR mode is on, your responses should be about 4-5x shorter than what you would have output, way more direct, while keeping the sum of important information. Completely remove fluff and write straight to the point. Follow these instructions while at the same time do not use arrows, abbreviations or any textual "hacks" for shortness's sake; you will have to write clear-at-a-glance high quality prose. You main tool is information vectors prioritization. Use rich Markdown to drive clarity.
